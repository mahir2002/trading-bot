## ⚙️ **CONFIGURATION**

### **Environment Variables (config.env)**

#### **Portfolio Settings**
```env
PORTFOLIO_BALANCE=10000          # Starting portfolio balance
MAX_POSITIONS=10                 # Maximum concurrent positions
POSITION_SIZE_PERCENT=5          # Position size as % of portfolio
```

#### **Trading Parameters**
```env
CONFIDENCE_THRESHOLD=70          # Minimum confidence for trades
TRADING_CYCLE_SECONDS=60         # Trading cycle interval
STOP_LOSS_PERCENT=5              # Stop loss percentage
TAKE_PROFIT_PERCENT=10           # Take profit percentage
```

#### **AI Model Settings**
```env
MODEL_RETRAIN_HOURS=24           # Model retraining frequency
TRAINING_SYMBOLS=15              # Number of symbols for training
LOOKBACK_DAYS=30                 # Historical data lookback
```

#### **Market Data**
```env
MIN_VOLUME_USDT=1000000          # Minimum volume filter
PRICE_CHANGE_THRESHOLD=2         # Price change threshold
```

#### **Dashboard Settings**
```env
DASHBOARD_HOST=0.0.0.0           # Dashboard host
DASHBOARD_PORT=8090              # Dashboard port
AUTO_REFRESH_SECONDS=10          # Auto-refresh interval
```

## 🛠️ **TROUBLESHOOTING**

### **Common Issues**

#### **Dashboard Not Loading**
```bash
# Check if port 8090 is available
netstat -an | grep 8090

# Try different port
python ultimate_unified_crypto_system.py --port 8091
```

#### **Missing Dependencies**
```bash
# Install missing packages
pip install -r requirements.txt

# Or use the launcher
python launch_ultimate_system.py
# Select option 5: Install Dependencies
```

#### **API Connection Issues**
- Check internet connection
- Verify API endpoints are accessible
- Review firewall settings
- Check rate limiting

#### **Performance Issues**
- Reduce refresh interval in config.env
- Limit number of tracked cryptocurrencies
- Close unused browser tabs
- Restart the system
