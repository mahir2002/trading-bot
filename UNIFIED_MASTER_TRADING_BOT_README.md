### 🚀 Quick Start Guide

#### **1. Installation**
```bash
# Install dependencies
python install_dependencies.py

# Or manually install
pip install ccxt pandas numpy ta scikit-learn requests python-dotenv
```

#### **2. Configuration**
```bash
# Copy and edit the configuration file
cp config.env.unified config.env

# Edit with your API keys
nano config.env
```

#### **3. System Check**
```bash
# Verify everything is working
python start_unified_bot.py --check-only
```

#### **4. Start Trading**
```bash
# Paper trading (recommended first)
python start_unified_bot.py --mode paper

# Live trading (use with caution)
python start_unified_bot.py --mode live

# Backtesting
python start_unified_bot.py --mode backtest
```
